Blockchain:
npx hardhat node
Em outro terminal: npx hardhat run --network localhost ignition/modules/deployERC20.js

API:
node server.js

FLUXOGRAMA
Crie uma rifa
http://localhost:3000/create-rifa
{
  "maxEntradas": 100,
  "valorEntrada": "10"
}
http://localhost:3000/approve
aprove essa rifa criada a gastar seus tokens
{
  "rifaAddress": "ENDERECO DA RIFA",
  "amount": "20"
}
entre na rifa
http://localhost:3000/enter-rifa
{
  "rifaAddress": "ENDERECO DA RIFA",
  "quantidadeRifas": 2
}

ARQUIVO .ENV
PRIVATE_KEY= private key da primeira conta que aparece no npx hardhat node
HARDHAT_RPC_URL=http://localhost:8545
CONTRACT_ADDRESS_REALDIGITAL=use o endereco ao dar deploy no contrato RealDigital.sol(npx hardhat run --network localhost ignition/modules/deployERC20.js)
